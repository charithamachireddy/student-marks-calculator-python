# Student Marks Calculator (Python)

Student Marks Calculator is a simple Python program that calculates a student's total marks, percentage, and grade based on marks entered for five subjects.


## Project Objective

The objective of this project is to practice basic Python programming concepts such as user input, arithmetic operations, and conditional statements.

## Features

- Accepts marks for 5 subjects
- Calculates total marks
- Calculates percentage
- Displays grade based on percentage
- Simple and beginner-friendly program


## Technologies Used

- Python
